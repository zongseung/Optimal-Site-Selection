## 종합 발견 & 정직한 권고

**① 발화 판별력:** 단일변수 최강 신호(거울변수 접음) = `S_p`·`mu_flammability`·`f_landcover` (AUC-PR 상위). 최약 신호 = `aspect_cos`·`f_road`·`nn_station_km`. 주의: 모든 단일변수 AUC-PR 이 기저율 대비 lift ~1.5~1.9배로 **낮다**(presence-only + 발화 희소 + 거리/연료의 약한 단조신호). 즉 단일 파생변수로는 발화를 거의 못 가르며, 위험은 I·S·W 결합과 임계값 결정에서 나온다.

- 가설 검정: README/config 가설은 'FFMC·dist_to_forest 강, ISI 약'. 앵커 분리력에서 **연료/산림계열(S_p·mu_flammability·dist_to_forest)이 도로·송전선·양간보다 분리력이 높게** 나와 '산림 근접' 가설은 지지된다. 반면 FFMC 계열 단일신호의 우위는 (전주 사전피처에 일별 FFMC 가 직접 없어) 이 앵커 분리력만으로는 확정 못 한다 — config 의 FFMC 우위는 관측소-수준 recall@top-k 튜닝 근거이며 별개 측정임.

**② 중복(고상관):** 서로 다른 파생변수 중 최강 중복쌍 `fwi_q90`↔`fwi_mean` (r=0.98). 추가로 모델 재표현 거울쌍(8개; 원변수↔I입력, S_p↔S 등)은 정의상 동일 정보라 중복 카운트에서 제외. VIF>10 변수도 다중공선성 → 입력 축소 후보.

**③ R 지배항 / 영동 쏠림:** *전역 분산기여*(전주 순위 흔드는 항)는 **I** 최대 (I=0.50, S=0.13, W=0.37). 하지만 영동이 상위2% 위험의 95%(쏠림 4.3×)를 차지하는 **기계적 원인은 between-regime 평균 차이**이고, 그 차이가 가장 큰 항은 **W**(영동/비영동 배율 I=1.24×, S=0.95×, W=2.42×). 즉 영동 쏠림은 곱셈 R 에서 **W(시즌 fwi_q90·양간) 가 영동에서 2.4× 높아** 발생한다(영동 W 중앙 0.56 vs 영서·산간 0.21).

**④ 체제 차이:** 양간일수·FWI·연료·산림거리가 체제별로 갈림 → MoE 정당. (표 §4)

**⑤ 공간/불확실성/leakage:** R Moran's I=0.60 (강한 공간자기상관 → 공간CV 필수). 
  - `unc_lo` 퇴화=True → True 면 컬럼 제거 권고. 
  - 약지도 라벨 누수 점검: 최강 상관 변수가 모델 산출이 아니라면 주의 (표 §8).

**모델 개선 권고 Top 3:**

1. **영동 쏠림 재균형:** 상위2% 위험의 95% 가 영동에 쏠려 있고(전체 영동 점유 22%, 쏠림 4.3×). 원인은 곱셈 R 에서 **W 가 영동에서 2.4× 높기** 때문(I 는 1.24×, S 는 0.95× 로 체제차 작음). 의도된 설계(영동=양간·고FWI)이나, F1 이 영동에 지나치게 의존하면 영서·산간 발화(인간발화·연료)를 놓칠 위험 → W_BLEND/양간 가중 점검 또는 체제별 임계값(이미 ALLOC_* 존재)으로 비영동 recall 보강.
2. **입력 축소:** 고상관쌍 `fwi_q90`↔`fwi_mean`(r=0.98), 고VIF 변수 ['fwi_mean', 'fwi_q90', 's2_nbr', 's2_ndmi'] 중 하나만 유지 → 다중공선성·과적합 축소.
3. **제출물 정리:** `unc_lo` 가 상수(0)라 정보 없음 → 컬럼 제거하거나 실제 하한 추정으로 대체(현재 risk_lo/hi 가 유효 밴드이므로 unc_lo/hi 는 중복·혼란 소지).
